**New Feature in Scripted Events V2.0.0**
***
The `SIZE` action grants player control in a script.

## Usage
```
SIZE <PLAYERS> <X> <Y> <Z>
```
Re-scales players to match the provided dimensions.

## Arguments
### Players
The players to resize.

### X
The x-dimension to size the player(s).

### Y
The x-dimension to size the player(s).

### Z
The x-dimension to size the player(s).
